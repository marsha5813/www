Jan. 21, 2017. Batch downloading files from a website

Okay, first I need to apologize for the clumsiness of this video tutorial. I do a bit of bumbling around, mostly because I didn't take a lot of time to prepare. I just turned on the recorder and went with it. During the video, I occasionally lose my mind and make simple coding mistakes. Don't judge me.

This video tutorial demonstrates how to do the following:

  • Set up a Linux Bash terminal on your Windows 10 PC.
  • Download all the links from a website (in this case, a list of .pdf files) automatically using a script I found on Github.

This tutorial assumes: 1) that you are using Windows 10; and 2) that you have no prior programming experience. I try to spell out in simple language what I'm doing, especially during the process of setting up the Bash terminal on your PC. So, what if my assumptions are violated? For instance, what if you're using a previous version of Windows? In that case, you can't run Linux natively on your machine. You need to use Cygwin or set up a virtual machine running Linux. If you're using a Mac, you're in luck! Mac OS can run Bash scripts directly from the terminal. And what if you have prior programming experience? Then you don't need this tutorial.

Finally, in case you're interested, here are all the .pdf files of Barack Obama's speeches used in this tutorial.

***UPDATE***
There are simpler ways to do the things in this tutorial. Other methods involve browser add-ons or some other software that you can click through with no coding involved. But by following the above tutorial, you'll learn other useful skills such as setting up Bash on your Windows PC and learning a few simple commands such as cloning a Git repository. If you think you'll ever use Github again for any reason, it's a useful exercise to follow my tutorial.
